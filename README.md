## Deploy

- Open a `pull-request`
- Merge the `pull-request` to `master` branch
- Travis will build the `master` branch and deploy it to `gh-pages` branch
 

## CI/CD with Netlify 

- Added Netlify to this `repo` 
- Every PR will now have its own Netlify url.
